= Cloner une machine virtuelle
:revdate: 2025-05-09
:page-revdate: {revdate}

Les machines virtuelles peuvent être clonées avec ou sans données. Cette fonction n'a pas besoin de prendre un instantané de la machine virtuelle ni de configurer d'abord une cible de sauvegarde.

== Cloner une machine virtuelle avec des données de volume

. Sur la page `Virtual Machines`, cliquez sur `Clone` des actions de la VM.
. Définissez un nouveau nom de machine virtuelle et cliquez sur `Create`.
+
image::vm/clone-vm-with-data.png[clone-vm-with-data.png]

== Cloner une machine virtuelle sans données de volume

Cloner une machine virtuelle sans données de volume crée une nouvelle machine virtuelle avec la même configuration que la source.

. Sur la page `Virtual Machines`, cliquez sur `Clone` des actions de la VM.
. Décochez la case `clone volume data`.
. Définissez un nouveau nom de machine virtuelle et cliquez sur `Create`.
+
image::vm/clone-vm-without-data.png[clone-vm-without-data.png]
