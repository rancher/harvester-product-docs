= Clone uma Máquina Virtual
:revdate: 2025-05-09
:page-revdate: {revdate}

Máquinas virtuais podem ser clonadas com ou sem dados. Esta função não precisa tirar um instantâneo da máquina virtual, nem configurar um destino de backup primeiro.

== Clone uma Máquina Virtual com Dados de Volume

. Na página `Virtual Machines`, clique em `Clone` das ações da VM.
. Defina um novo nome para a máquina virtual e clique em `Create`.
+
image::vm/clone-vm-with-data.png[clone-vm-with-data.png]

== Clone uma Máquina Virtual sem Dados de Volume

Clonar uma máquina virtual sem dados de volume cria uma nova máquina virtual com a mesma configuração da origem.

. Na página `Virtual Machines`, clique em `Clone` das ações da VM.
. Desmarque a caixa de seleção `clone volume data`.
. Defina um novo nome para a máquina virtual e clique em `Create`.
+
image::vm/clone-vm-without-data.png[clone-vm-without-data.png]
