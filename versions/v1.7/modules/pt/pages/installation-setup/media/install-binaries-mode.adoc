= Instalar apenas binários
:revdate: 2025-05-09
:page-revdate: {revdate}

Você pode instalar e configurar apenas binários, o que é ideal para casos de uso em nuvem e edge.

image::install/choose-installation-mode.png[choose-installation-mode.png]

== Background

Atualmente, quando um novo nó é lançado, ele precisa ser o primeiro nó no cluster ou se juntar a um cluster existente.
Esses dois modos são úteis quando você já sabe o suficiente sobre o ambiente para instalar o nó.
No entanto, para casos de uso como provedores de nuvem bare metal e o edge, esses modos de instalação carregam o SO e o conteúdo para o nó sem permitir que você configure a rede. Além disso, a configuração do K8s e da rede não será aplicada.

Se você escolher `Install Harvester binaries only`, precisará realizar configurações adicionais após a primeira inicialização:

* Opção Criar/Entrar
* Detalhes da interface de rede de gerenciamento
* Token do cluster
* Senha do nó

Então, o instalador aplicará a configuração do endpoint e inicializará {harvester-product-name}. Nenhuma reinicialização adicional será necessária.

== Modo de disco em streaming

A SUSE publicou um artefato de imagem bruta para pré-instalado {harvester-product-name}. O instalador agora permite transmitir uma imagem pré-instalada diretamente para o disco para suportar melhor a integração com provedores de nuvem.

Em `Equinix Metal`, você pode usar os seguintes argumentos do kernel para usar o modo de streaming:

----
ip=dhcp net.ifnames=1 rd.cos.disable rd.noverifyssl root=live:http://${artifactEndpoint}/harvester-v1.2.0-rootfs-amd64.squashfs harvester.install.automatic=true harvester.scheme_version=1 harvester.install.device=/dev/vda  harvester.os.password=password harvester.install.raw_disk_image_path=http://${artifactEndpoint}/harvester-v1.2.0-amd64.raw harvester.install.mode=install console=tty1 harvester.install.tty=tty1 harvester.install.config_url=https://metadata.platformequinix.com/userdata harvester.install.management_interface.interfaces="name:enp1s0" harvester.install.management_interface.method=dhcp harvester.install.management_interface.bond_options.mode=balance-tlb harvester.install.management_interface.bond_options.miimon=100
----

[NOTE]
====
Ao transmitir para o disco, é recomendável hospedar o artefato de disco bruto mais próximo dos alvos, pois o artefato de disco bruto tem quase 16G de tamanho.
====
