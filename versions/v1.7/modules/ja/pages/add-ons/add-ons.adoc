= アドオン
:revdate: 2025-12-30
:page-revdate: {revdate}

{harvester-product-name} は、最小限のインストールフットプリントを維持するためのオプション機能をアドオンとして提供します。この設計により、ユースケースや要件に基づいて機能を有効または無効にすることができます。各アドオンの設定をカスタマイズできますが、カスタマイズのレベルは基盤となるソフトウェアに依存します。

このバージョンで利用可能なアドオンは次のとおりです：

* xref:add-ons/lvm-local-storage.adoc[csi-driver-lvm]（実験的）
* xref:add-ons/kubeovn-operator.adoc[kubeovn-operator]（実験的）
* xref:add-ons/nvidia-driver-toolkit.adoc[nvidia-driver-toolkit]
* xref:add-ons/pcidevices-controller.adoc[pcidevices-controller]
* xref:observability/logging.adoc[rancher-logging]
* xref:observability/monitoring.adoc[rancher-monitoring]
* xref:add-ons/rancher-vcluster.adoc[rancher-vcluster]（実験的）
* xref:add-ons/harvester-seeder.adoc[seeder]（実験的）
* xref:add-ons/vm-dhcp-controller.adoc[vm-dhcp-controller]（実験的）
* xref:add-ons/vm-import-controller.adoc[vm-import-controller]

{harvester-product-name} UIの *Add-ons* 画面でアドオンを有効または無効にできます。

|===
| 操作 | アクション | 新しい状態

| アドオンを有効にする
| アドオンを選択し、次に *⋮ -> 有効にする* を選択します。
| **DeploySuccessful**

| アドオンを無効にする
| アドオンを選択し、次に *⋮ -> 無効にする* を選択します。
| *無効化済み*
|===

[NOTE]
====
{harvester-product-name} は無効化されたアドオンの設定データを保存し、再有効化時に即座に再利用できるようにします。
====
