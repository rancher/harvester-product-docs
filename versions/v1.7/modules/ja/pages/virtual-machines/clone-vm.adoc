= 仮想マシンをクローンします。
:revdate: 2025-05-09
:page-revdate: {revdate}

仮想マシンはデータの有無にかかわらずクローンできます。この機能は、仮想マシンのスナップショットを取得したり、バックアップ先を設定したりする必要はありません。

== ボリュームデータを持つ仮想マシンをクローンします。

. `Virtual Machines`ページで、VMアクションの`Clone`をクリックします。
. 新しい仮想マシン名を設定し、`Create`をクリックします。
+
image::vm/clone-vm-with-data.png[clone-vm-with-data.png]

== ボリュームデータなしで仮想マシンをクローンします。

ボリュームデータなしで仮想マシンをクローンすると、元の仮想マシンと同じ設定の新しい仮想マシンが作成されます。

. `Virtual Machines`ページで、VMアクションの`Clone`をクリックします。
. `clone volume data`チェックボックスの選択を解除します。
. 新しい仮想マシン名を設定し、`Create`をクリックします。
+
image::vm/clone-vm-without-data.png[clone-vm-without-data.png]
