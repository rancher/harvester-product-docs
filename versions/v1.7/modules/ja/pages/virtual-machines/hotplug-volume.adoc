= ホットプラグボリューム
:revdate: 2025-05-09
:page-revdate: {revdate}

{harvester-product-name}は、実行中の仮想マシンにホットプラグボリュームを追加することをサポートしています。

[IMPORTANT]
====
現在、KubeVirtはホットプラグボリューム用にディスクバス`scsi`のみをサポートしています。詳細については、この https://github.com/kubevirt/kubevirt/issues/5080#issuecomment-785183128[問題]を参照してください。
====

== 実行中の仮想マシンにホットプラグボリュームを追加する

以下の手順は、実行中の仮想マシンと準備が整ったボリュームがあることを前提としています：

. *仮想マシン*ページに移動します。
. ボリュームを追加したい仮想マシンを見つけて、*⋮ -> ボリュームを追加*を選択します。
+
image::vm/add-volume-button.png[ボリューム追加ボタン]
+
. *名前*を入力し、*ボリューム*を選択します。
. *適用*をクリックします。
+
image::vm/add-volume-panel.png[ボリューム追加パネル]
