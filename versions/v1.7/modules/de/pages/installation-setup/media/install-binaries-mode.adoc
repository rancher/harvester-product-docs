= Installieren Sie nur Binärdateien
:revdate: 2025-05-09
:page-revdate: {revdate}

Sie können nur Binärdateien installieren und konfigurieren, was ideal für Cloud- und Edge-Anwendungsfälle ist.

image::install/choose-installation-mode.png[choose-installation-mode.png]

== Hintergrund

Derzeit muss ein neuer Knoten, wenn er gestartet wird, der erste Knoten im Cluster sein oder einem bestehenden Cluster beitreten.
Diese beiden Modi sind nützlich, wenn Sie bereits genug über die Umgebung wissen, um den Knoten zu installieren.
Für Anwendungsfälle wie Bare-Metal-Cloud-Anbieter und Edge laden diese Installationsmodi jedoch das Betriebssystem und den Inhalt auf den Knoten, ohne dass Sie das Netzwerk konfigurieren können. Darüber hinaus wird die K8s- und Netzwerk-Konfiguration nicht angewendet.

Wenn Sie `Install Harvester binaries only` wählen, müssen Sie nach dem ersten Bootvorgang zusätzliche Konfiguration durchführen:

* Erstellen/Beitreten-Option
* Details zur Verwaltungsnetzwerkschnittstelle
* Cluster-Token
* Knoten-Passwort

Dann wird der Installer die Endpunktkonfiguration anwenden und {harvester-product-name} starten. Es sind keine weiteren Neustarts erforderlich.

== Stream-Disk-Modus

SUSE hat ein Rohbild-Artefakt für vorinstallierte {harvester-product-name} veröffentlicht. Der Installer ermöglicht jetzt das Streaming eines vorinstallierten Images direkt auf die Festplatte, um eine bessere Integration mit Cloud-Anbietern zu unterstützen.

Auf `Equinix Metal` können Sie die folgenden Kernel-Argumente verwenden, um den Streaming-Modus zu aktivieren:

----
ip=dhcp net.ifnames=1 rd.cos.disable rd.noverifyssl root=live:http://${artifactEndpoint}/harvester-v1.2.0-rootfs-amd64.squashfs harvester.install.automatic=true harvester.scheme_version=1 harvester.install.device=/dev/vda  harvester.os.password=password harvester.install.raw_disk_image_path=http://${artifactEndpoint}/harvester-v1.2.0-amd64.raw harvester.install.mode=install console=tty1 harvester.install.tty=tty1 harvester.install.config_url=https://metadata.platformequinix.com/userdata harvester.install.management_interface.interfaces="name:enp1s0" harvester.install.management_interface.method=dhcp harvester.install.management_interface.bond_options.mode=balance-tlb harvester.install.management_interface.bond_options.miimon=100
----

[NOTE]
====
Beim Streaming auf die Festplatte wird empfohlen, das Rohbild-Artefakt näher an den Zielen zu hosten, da das Rohbild-Artefakt fast 16 G groß ist.
====
