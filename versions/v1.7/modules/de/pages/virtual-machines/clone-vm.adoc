= Virtuelle Maschine klonen
:revdate: 2025-05-09
:page-revdate: {revdate}

Virtuelle Maschinen können mit oder ohne Daten geklont werden. Diese Funktion erfordert weder die Erstellung eines Snapshots der virtuellen Maschine noch die Einrichtung eines Backup-Ziels.

== Virtuelle Maschine mit Volumendaten klonen

. Klicken Sie auf der `Virtual Machines`-Seite auf `Clone` der VM-Aktionen.
. Geben Sie einen neuen Namen für die virtuelle Maschine ein und klicken Sie auf `Create`.
+
image::vm/clone-vm-with-data.png[clone-vm-with-data.png]

== Virtuelle Maschine ohne Volumendaten klonen

Das Klonen einer virtuellen Maschine ohne Volumendaten erstellt eine neue virtuelle Maschine mit derselben Konfiguration wie die Quelle.

. Klicken Sie auf der `Virtual Machines`-Seite auf `Clone` der VM-Aktionen.
. Entfernen Sie das Häkchen im `clone volume data`-Kontrollkästchen.
. Geben Sie einen neuen Namen für die virtuelle Maschine ein und klicken Sie auf `Create`.
+
image::vm/clone-vm-without-data.png[clone-vm-without-data.png]
