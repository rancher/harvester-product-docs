= Exportieren Sie ein Volume in ein Bild
:revdate: 2025-05-09
:page-revdate: {revdate}

Sie können ein vorhandenes Volume in ein Bild exportieren, indem Sie die folgenden Schritte ausführen:

[NOTE]
====
Wenn das Volume auf externem Speicher ist, stellen Sie sicher, dass es nicht an Arbeitslasten (virtuellen Maschinen und Pods) angeschlossen ist, bevor Sie den Exportvorgang starten. Dies verhindert, dass das resultierende Bild im *Exporting*-Zustand hängen bleibt.
====

. Klicken Sie auf die `⋮` Schaltfläche und wählen Sie die `Export Image` Option aus.
+
image::volume/export-volume-to-image-1.png[export-volume-to-image-1]

. Wählen Sie das `Namespace` des neuen Bildes aus.
. Konfigurieren Sie das `Name` des neuen Bildes.
. Wählen Sie ein vorhandenes `StorageClass` aus.
. (Optional) Sie können das exportierte Bild von der `Images` Seite herunterladen, indem Sie auf die `⋮` Schaltfläche klicken und die `Download` Option auswählen.
+
image::volume/export-volume-to-image-2.png[export-volume-to-image-2]
