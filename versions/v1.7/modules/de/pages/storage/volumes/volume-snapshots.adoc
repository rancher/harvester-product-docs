= Volume Snapshots
:revdate: 2025-05-09
:page-revdate: {revdate}

Ein Volume Snapshot stellt einen Snapshot eines Volumes in einem Speichersystem dar. Nachdem Sie ein Volume erstellt haben, können Sie einen Volume Snapshot erstellen und ein Volume auf den Zustand des Snapshots zurücksetzen. Mit Volume Snapshots können Sie die Konfiguration eines Volumes einfach kopieren oder wiederherstellen.

== Volume Snapshots erstellen

Sie können einen Volume Snapshot von einem vorhandenen Volume erstellen, indem Sie die folgenden Schritte ausführen:

. Gehen Sie zur Seite *Volumes*.
. Wählen Sie das Volume aus, von dem Sie einen Snapshot erstellen möchten, und wählen Sie *⋮ > Snapshot erstellen*.
+
image::volume/create-volume-snapshot-1.png[create-volume-snapshot-1]

. Geben Sie einen *Namen* für den Snapshot ein.
+
image::volume/create-volume-snapshot-2.png[create-volume-snapshot-2]

. Wählen Sie *Erstellen*, um einen neuen Volume Snapshot zu erstellen.
. Überprüfen Sie den Status dieser Operation und sehen Sie sich alle Volume Snapshots an, indem Sie zur *Volumes*-Seite gehen und die Registerkarte *Snapshots* auswählen. Wenn der *Ready To Use* den Status *√* erreicht, ist der Volume Snapshot bereit zur Verwendung.

[NOTE]
====
Ein wiederkehrender Snapshot wird derzeit nicht unterstützt und wird über https://github.com/harvester/harvester/issues/572[harvester/harvester#572] verfolgt.
====


== Ein neues Volume aus einem Volume Snapshot wiederherstellen

Sie können ein neues Volume aus einem vorhandenen Volume Snapshot wiederherstellen, indem Sie die folgenden Schritte ausführen:

. Gehen Sie zur Seite **Sicherung & Snapshot > Volume Snapshots** oder wählen Sie ein *Volume* von der Seite *Volumes* aus und gehen Sie zur Registerkarte *Snapshots*.
. Wählen Sie *⋮ > Wiederherstellen*.
+
image::volume/restore-volume-snapshot-1.png[restore-volume-snapshot-1]
+
image::volume/restore-volume-snapshot-2.png[restore-volume-snapshot-2]

. Geben Sie den *Namen* des neuen Volumes an.
+
image::volume/restore-volume-snapshot-3.png[restore-volume-snapshot-3]

. Wenn das Quellvolumen kein Bildvolumen ist, können Sie eine andere *Speicherklasse* auswählen. Sie können die *Speicherklasse* nicht ändern, wenn das Quellvolumen ein Bildvolumen ist.
+
image::volume/restore-volume-snapshot-4.png[restore-volume-snapshot-4]

. Wählen Sie *Erstellen*, um die Wiederherstellung eines neuen Volumens abzuschließen.
