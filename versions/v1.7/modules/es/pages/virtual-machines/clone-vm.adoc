= Clonar una máquina virtual
:revdate: 2025-05-09
:page-revdate: {revdate}

Las máquinas virtuales se pueden clonar con o sin datos. Esta función no necesita tomar una instantánea de la máquina virtual ni configurar primero un destino para la copia de seguridad.

== Clonar una máquina virtual con datos de volumen

. En la página `Virtual Machines`, haz clic en `Clone` de las acciones de la VM.
. Establece un nuevo nombre para la máquina virtual y haz clic en `Create`.
+
image::vm/clone-vm-with-data.png[clone-vm-with-data.png]

== Clonar una máquina virtual sin datos de volumen

Clonar una máquina virtual sin datos de volumen crea una nueva máquina virtual con la misma configuración que la fuente.

. En la página `Virtual Machines`, haz clic en `Clone` de las acciones de la VM.
. Desmarca la casilla `clone volume data`.
. Establece un nuevo nombre para la máquina virtual y haz clic en `Create`.
+
image::vm/clone-vm-without-data.png[clone-vm-without-data.png]
