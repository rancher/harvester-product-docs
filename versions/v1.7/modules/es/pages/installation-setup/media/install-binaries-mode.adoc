= Instalar solo binarios
:revdate: 2025-05-09
:page-revdate: {revdate}

Solo puedes instalar y configurar binarios, lo cual es ideal para casos de uso en la nube y en el edge.

image::install/choose-installation-mode.png[choose-installation-mode.png]

== Fondo

Actualmente, cuando se lanza un nuevo nodo, debe ser el primer nodo en el clúster o unirse a un clúster existente.
Estos dos modos son útiles cuando ya conoces lo suficiente sobre el entorno para instalar el nodo.
Sin embargo, para casos de uso como los proveedores de nube de equipo sin sistema operativo y el edge, estos modos de instalación cargan el sistema operativo y el contenido al nodo sin permitirte configurar la red. Además, la configuración de K8s y de red no se aplicará.

Si eliges `Install Harvester binaries only`, necesitarás realizar una configuración adicional después del primer arranque:

* Opción Crear/Unir
* Detalles de la interfaz de red de gestión
* Token del clúster
* Contraseña del nodo

Luego, el instalador aplicará la configuración del punto final y arrancará {harvester-product-name}. No se requerirán más reinicios.

== Modo de disco en streaming

SUSE ha publicado un artefacto de imagen en bruto para preinstalado {harvester-product-name}. El instalador ahora permite transmitir una imagen preinstalada directamente al disco para soportar una mejor integración con proveedores de nube.

En `Equinix Metal`, puedes usar los siguientes argumentos del núcleo de Linux para utilizar el modo de streaming:

----
ip=dhcp net.ifnames=1 rd.cos.disable rd.noverifyssl root=live:http://${artifactEndpoint}/harvester-v1.2.0-rootfs-amd64.squashfs harvester.install.automatic=true harvester.scheme_version=1 harvester.install.device=/dev/vda  harvester.os.password=password harvester.install.raw_disk_image_path=http://${artifactEndpoint}/harvester-v1.2.0-amd64.raw harvester.install.mode=install console=tty1 harvester.install.tty=tty1 harvester.install.config_url=https://metadata.platformequinix.com/userdata harvester.install.management_interface.interfaces="name:enp1s0" harvester.install.management_interface.method=dhcp harvester.install.management_interface.bond_options.mode=balance-tlb harvester.install.management_interface.bond_options.miimon=100
----

[NOTE]
====
Al transmitir al disco, se recomienda alojar el artefacto de disco en bruto más cerca de los objetivos, ya que el artefacto de disco en bruto tiene casi 16G de tamaño.
====
