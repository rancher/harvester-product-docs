= Extensión de Harvester UI
:revdate: 2025-12-30
:page-revdate: {revdate}

Las versiones {rancher-short-name} 2.10.0 y posteriores se integran con {harvester-product-name} utilizando la https://github.com/harvester/harvester-ui-extension[Extensión de Harvester UI], que se basa en la biblioteca https://www.npmjs.com/package/@rancher/shell[@rancher/shell].

[NOTE]
====
La interfaz de usuario heredada {harvester-product-name} sigue siendo compatible con v1.3.x y v1.4.x. Las nuevas características en versiones posteriores se implementan en la Extensión de Harvester UI.
====

== Matriz de soporte

La Extensión de Harvester UI es compatible con versiones anteriores de {harvester-product-name}. Utiliza banderas de características para mostrar las funciones de la interfaz de usuario que corresponden a la versión del clúster {harvester-product-name}.

Las versiones del sistema operativo {harvester-product-name} y la Extensión de Harvester UI se alinean a partir de v1.5.0. Para acceder a nuevas funciones de la interfaz de usuario, instala la última Extensión de Harvester UI en tu instancia de {rancher-short-name}.

|===
| Versión de la Extensión de Harvester UI | Versiones compatibles de {rancher-short-name}

| v1.0.x
| 2.10.1 y posteriores

| v1.5.x
| 2.11.0 y posteriores

| v1.6.x
| 2.12.0 y posteriores

| v1.7.x
| 2.13.0 y posteriores
|===

Instalar la extensión a través de la red no es posible en entornos aislados, por lo que debes realizar la solución alternativa descrita en xref:../../installation-setup/airgap.adoc#_harvester_ui_extension_with_rancher_integration[Extensión de Harvester UI con integración de {rancher-short-name}].

== Instalación en {rancher-short-name} 2.10

=== Instalación automática

Puedes instalar la Extensión de Harvester UI con un solo clic.

. En la interfaz de usuario de {rancher-short-name}, ve a *Gestión de Virtualización*.
+
image::rancher/auto-install-ui-extension.png[]

. En la pestaña *Clústeres de Harvester*, haz clic en *Instalar*.
+
Permite un tiempo para que la extensión se instale. La pantalla se actualiza automáticamente una vez que se completa la instalación.
+
[NOTE]
====
No navegues lejos de la pantalla durante el proceso de instalación.
====

=== Instalación manual

Si la instalación automática falla, puedes especificar la URL del repositorio de la extensión de la interfaz de usuario de Harvester y luego instalar la extensión.

image::rancher/ui-extension-install-failed.png[]

. En la interfaz de usuario de {rancher-short-name}, ve a *local -> Aplicaciones -> Repositorios*, y luego haz clic en *Crear*.
+
image::rancher/ui-extension-app-repository-setup.png[]

. Configure los ajustes siguientes:
+
* Nombre: Especifica un nombre para el repositorio.
* Destino: Selecciona *Repositorio Git que contiene definiciones de gráficos Helm o plantillas de clúster*.
* URL del Repositorio Git: Especifica *https://github.com/harvester/harvester-ui-extension*.
* Rama de Git: Especifica *gh-pages*.

. Haga clic en *Crear*.

. Ve a *Extensiones -> Disponibles*.
+
image::rancher/ui-extension-available-tab.png[]

. Localiza la extensión llamada *Harvester*, y luego haz clic en *Instalar*.

== Actualizaciones

=== Actualizaciones Automáticas

El botón *Actualizar* aparece siempre que hay una nueva versión de la extensión disponible.

. En la interfaz de usuario de {rancher-short-name}, ve a *Gestión de Virtualización*.
+
image::rancher/ui-extension-update.png[]

. Haz clic en *Actualizar*.
+
Permite un tiempo para que la extensión se actualice. La pantalla se actualiza automáticamente una vez que se completa la actualización.

=== Actualizaciones manuales

. En la interfaz de usuario de {rancher-short-name}, ve a *local -> Aplicaciones -> Repositorios*.

. Localiza el repositorio con la siguiente configuración y luego selecciona *⋮ -> Actualizar*.
+
* URL: *https://github.com/harvester/harvester-ui-extension*
* Rama: *gh-pages*
+
image::upgrade/rancher-2.10.1-repository-page.png[]

. Ve a *Extensiones -> Instaladas*.

. Localiza la extensión llamada *Harvester* y luego haz clic en *Actualizar*.

. Selecciona la nueva versión y luego haz clic en *Actualizar*.

image::upgrade/update-harvester-ui-extension-modal.png[]

. Permite un tiempo para que la extensión se actualice y, a continuación, actualiza la pantalla.

== limitaciones

=== Las versiones {harvester-product-name} v1.2.x y anteriores no son compatibles.

Cuando importas un clúster que está ejecutando {harvester-product-name} v1.2.x o anteriores, aparece un mensaje de error y el enlace del clúster se desactiva.

El botón **Gestionar** sigue disponible en {rancher-short-name} 2.11.0, pero se desactivará en una futura versión. Para obtener más información, consulta el problema https://github.com/harvester/harvester/issues/8054[#8054].

[CAUTION]
====
Evita usar el botón *Gestionar* en la interfaz de usuario de {rancher-short-name} si tus clústeres están ejecutando {harvester-product-name} v1.2.x o anteriores. Gestionar clústeres utilizando esta función puede resultar en un comportamiento inesperado.
====
