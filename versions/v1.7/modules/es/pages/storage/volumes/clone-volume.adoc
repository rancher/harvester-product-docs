== Instrucciones para clonar un volumen

Después de crear un volumen, puedes clonar el volumen siguiendo los pasos a continuación:

. Haz clic en el botón `⋮` y selecciona la opción `Clone`.
+
image::volume/clone-volume-1.png[clone-volume-1]

. Seleccione `clone volume data`.
. Configura el `Name` del nuevo volumen y haz clic en `Create`.
. (Opcional) Un volumen clonado puede añadirse a una máquina virtual utilizando `Add Existing Volume`.
+
image::volume/clone-volume-2.png[clone-volume-2]

=== Reconstrucción de réplicas para volúmenes clonados {longhorn-product-name} desconectados

Cuando clonas un volumen desconectado {longhorn-product-name} en un clúster de múltiples nodos, {longhorn-product-name} crea solo _una réplica_ en lugar del número esperado (por ejemplo, tres réplicas en un clúster de tres nodos). El estado del clon de volumen mostrado es `copy-completed-awaiting-healthy` en lugar de `complete`.

{longhorn-product-name} reconstruye automáticamente las réplicas restantes después de que el volumen clonado se adjunta a una máquina virtual. Sin embargo, puedes iniciar el proceso de reconstrucción de réplicas inmediatamente para el volumen desconectado realizando cualquiera de las siguientes acciones:

* En la interfaz de usuario de {longhorn-product-name}, adjunta manualmente el volumen y espera a que se complete el proceso de reconstrucción de réplicas.

* Habilita la configuración {longhorn-product-name} https://documentation.suse.com/cloudnative/storage/latest/en/high-availability/offline-replica-rebuilding.html[`offline-replica-rebuilding`] para permitir la reconstrucción automática de réplicas para volúmenes desconectados.

[NOTE]
====
Este comportamiento se aplica exclusivamente a volúmenes desconectados. {longhorn-product-name} realiza automáticamente la reconstrucción de réplicas para volúmenes que están adjuntos a una máquina virtual en funcionamiento.
====
