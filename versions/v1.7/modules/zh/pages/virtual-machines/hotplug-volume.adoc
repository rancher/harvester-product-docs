= 热插拔卷
:revdate: 2025-05-09
:page-revdate: {revdate}

{harvester-product-name} 支持向正在运行的虚拟机添加热插拔卷。

[IMPORTANT]
====
目前，KubeVirt 仅支持热插拔卷的磁盘总线 `scsi`。有关更多信息，请参见此 https://github.com/kubevirt/kubevirt/issues/5080#issuecomment-785183128[问题]。
====

== 向正在运行的虚拟机添加热插拔卷

以下步骤假设您有一个正在运行的虚拟机和一个准备好的卷：

. 转到 *虚拟机* 页面。
. 找到您想要添加卷的虚拟机并选择 *⋮ -> 添加卷*。
+
image::vm/add-volume-button.png[添加卷按钮]
+
. 输入 *名称* 并选择 *卷*。
. 单击 *应用*。
+
image::vm/add-volume-panel.png[添加卷面板]
