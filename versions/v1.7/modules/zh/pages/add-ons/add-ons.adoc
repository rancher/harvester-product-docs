= 附加组件
:revdate: 2025-12-30
:page-revdate: {revdate}

{harvester-product-name} 提供可选功能作为附加产品，以保持最小的安装占用空间。此设计允许您根据使用案例和需求启用或禁用功能。您可以自定义每个附加产品的设置，但自定义的程度取决于底层软件。

此版本提供以下附加产品：

* xref:add-ons/lvm-local-storage.adoc[csi-driver-lvm]（实验性）
* xref:add-ons/kubeovn-operator.adoc[kubeovn-operator]（实验性）
* xref:add-ons/nvidia-driver-toolkit.adoc[nvidia-driver-toolkit]
* xref:add-ons/pcidevices-controller.adoc[pcidevices-controller]
* xref:observability/logging.adoc[rancher-logging]
* xref:observability/monitoring.adoc[rancher-monitoring]
* xref:add-ons/rancher-vcluster.adoc[rancher-vcluster]（实验性）
* xref:add-ons/harvester-seeder.adoc[seeder]（实验性）
* xref:add-ons/vm-dhcp-controller.adoc[vm-dhcp-controller]（实验性）
* xref:add-ons/vm-import-controller.adoc[vm-import-controller]

您可以在 *附加产品* 屏幕的 {harvester-product-name} 界面上启用和禁用附加产品。

|===
| Operation | 操作 | 新状态

| 启用附加产品
| 选择附加产品，然后选择 *⋮ -> 启用*。
| **部署成功**

| 禁用附加产品
| 选择附加产品，然后选择 *⋮ -> 禁用*。
| *已禁用*
|===

[NOTE]
====
{harvester-product-name} 存储禁用附加产品的配置数据，允许在重新启用时立即重用。
====
