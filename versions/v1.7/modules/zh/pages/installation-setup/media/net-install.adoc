= 网络安装 ISO
:revdate: 2025-05-09
:page-revdate: {revdate}

网络安装 ISO 是一个最小化的安装映像，仅包含操作系统内核组件，允许安装程序启动并在磁盘上安装操作系统。安装完成后，操作系统会从互联网（主要是 Docker Hub）拉取所有所需的容器映像。

您可以在以下情况下使用网络安装 ISO：

* 服务器上的虚拟媒体实现存在缺陷或运行缓慢。一些用户报告说，ISO 重定向速度太慢，无法将所有映像预加载到系统中。有关更多信息，请参见 https://github.com/harvester/harvester/issues/2651[问题2651]。
* 您有一个私有注册表，其中包含所有{harvester-product-name}映像，以及配置 containerd 镜像所需的知识和经验。

[CAUTION]
====
*您必须始终使用完整的 ISO 来启动 {harvester-product-name} 集群*（换句话说，使用没有 `-net-install` 后缀的 ISO）。完整的 ISO 包含所有所需的映像，安装程序在安装过程中预加载这些映像。使用网络安装 ISO 启动集群时，您可能会轻松达到 https://docs.docker.com/docker-hub/download-rate-limit/[Docker Hub 速率限制]。
====

== 用法

从 GitHub https://github.com/harvester/harvester/releases[发布] 页面下载网络安装 ISO，然后启动 ISO 以安装 {harvester-product-name}。网络安装 ISO 映像文件名的后缀为 `net-install`（例如， https://releases.rancher.com/harvester/v1.3.0/harvester-v1.3.0-amd64-net-install.iso).）

== PXE 安装

如果您决定使用网络安装 ISO 作为 PXE 安装源，请在启动内核时添加以下参数：

----
harvester.install.with_net_images=true
----

有关更多信息，请参见xref:../../installation-setup/methods/pxe-boot-install.adoc[PXE 引导安装]。
