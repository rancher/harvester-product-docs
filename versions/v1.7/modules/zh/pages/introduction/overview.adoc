= 概述
:revdate: 2025-07-16
:page-revdate: {revdate}

{harvester-product-name-tm} 是一款基于 Kubernetes 构建的现代化、可互操作的 https://en.wikipedia.org/wiki/Hyper-converged_infrastructure[超融合基础架构 (HCI)] 解决方案。它是一个开源替代方案，旨在为寻求 https://about.gitlab.com/topics/cloud-native/[云原生] HCI 解决方案的操作员提供服务。{harvester-product-name} 在裸机服务器上运行，并提供集成的虚拟化和分布式存储功能。除了传统的虚拟机 (VM) 之外，{harvester-product-name} 还通过与 https://documentation.suse.com/cloudnative/rancher-manager/v2.11/en/integrations/harvester/harvester.html[{rancher-product-name-tm}] 的集成，自动支持容器化环境。它提供统一传统虚拟化基础架构的解决方案，同时支持从核心数据中心到边缘站点全面落地容器技术。

== 体系结构

该架构由尖端的开源技术组成：

* *Linux 操作系统。* https://documentation.suse.com/cloudnative/os-manager/[{elemental-product-name}] 是 SUSE Linux Enterprise Micro 的核心，{harvester-product-name}是一个不可变的 Linux 发行版，旨在尽可能减少 Kubernetes 集群中的操作系统维护。
* *构建在 Kubernetes 之上。* https://kubernetes.io/[Kubernetes] 已成为所有形式因素中占主导地位的基础设施语言，而 {harvester-product-name} 是一个以 Kubernetes 为基础的 HCI 解决方案。
* *使用 KubeVirt 进行虚拟化管理。* https://kubevirt.io/[KubeVirt] 通过在 Kubernetes 之上利用 KVM 提供虚拟化管理。
* *使用 {longhorn-product-name} 进行存储管理。* https://documentation.suse.com/cloudnative/storage/1.8/en/longhorn-documentation.html[{longhorn-product-name}] 提供分布式块存储和分层。
* *使用 Grafana 和 Prometheus 进行可观测性。* https://grafana.com/[Grafana] 和 https://prometheus.io/[Prometheus] 提供强大的监控和日志记录。

== 功能

{harvester-product-name} 是一个企业级、易于使用的基础设施平台，利用本地直接附加存储，而不是复杂的外部 SAN。它利用 Kubernetes API 作为容器和虚拟机工作负载的统一自动化语言。一些主要特性包括：

* *易于入门。*由于 {harvester-product-name} 作为可启动的设备映像发布，您可以直接在裸机服务器上安装它，使用 https://github.com/harvester/harvester/releases[ISO 映像] 或通过 xref:../installation-setup/methods/pxe-boot-install.adoc[iPXE] 脚本自动安装。
* *虚拟机生命周期管理。*轻松创建、编辑、克隆和删除虚拟机，包括 SSH 密钥注入、cloud-init 以及图形和串口控制台。
* *虚拟机实时迁移。*将虚拟机移动到不同的主机或节点，零停机时间。
* *虚拟机备份、快照和恢复。*从 NFS、S3 服务器或 NAS 设备备份您的虚拟机。使用备份恢复失败的虚拟机或在不同的集群上创建新的虚拟机。
* *存储管理。* {harvester-product-name} 支持分布式块存储和分层。卷代表存储；您可以轻松创建、编辑、克隆或导出卷。
* *网络管理。*支持使用虚拟IP（VIP）和多个网络接口（NIC）。如果您的虚拟机需要连接到外部网络，请创建VLAN或无标签网络。
* *与 {rancher-product-name} 的集成。*通过 {harvester-product-name} 的虚拟化管理页面直接访问 {rancher-product-name}，并与您的 Kubernetes 集群一起管理虚拟机工作负载。

== 用户界面

{harvester-product-name} 提供强大且易于使用的基于 Web 的界面，用于可视化和管理您的基础设施。安装后，您可以从节点的终端访问 UI 的 IP 地址。
