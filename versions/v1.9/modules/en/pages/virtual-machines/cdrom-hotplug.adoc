= CD-ROM Hotplug
:revdate: 2026-06-02
:page-revdate: {revdate}

{harvester-product-name} supports hotplugging and hotunplugging CD-ROM volumes for running virtual machines. This feature allows you to insert and eject images from CD-ROM devices.

[NOTE]
====
CD-ROM volume hotplugging is supported only for CD-ROM devices of virtual machines created in {harvester-product-name} v1.8.0 or later.

This feature applies only to volume data inside an existing CD-ROM device. Attaching or detaching the actual CD-ROM device still requires a virtual machine restart.
====

== Ejecting an image from a CD-ROM device

. On the {harvester-product-name} UI, go to the *Virtual Machines* screen.

. Click the name of the target virtual machine, and then go to the *Volumes* tab.
+
image::vm/eject-image.png[Ejecting an image from a CD-ROM device]

. Locate the target CD-ROM volume, and then click *Eject Image*.

. Click *Apply*.

== Inserting an image into an empty CD-ROM device

. On the {harvester-product-name} UI, go to the *Virtual Machines* screen.

. Click the name of the target virtual machine, and then go to the *Volumes* tab.
+
image::vm/insert-image.png[Inserting an image into an empty CD-ROM device]

. Locate the target CD-ROM volume, and then click *Insert Image*.

. Select the image, and then click *Apply*.
+
image::vm/insert-image-dialog.png[Selecting the image to be inserted]
